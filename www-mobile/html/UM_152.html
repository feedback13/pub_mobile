오투잡 파워셀러 인터뷰

메인 컨텐츠

어떻게 재능을 판매하게 됐나요? 재능을 판매하게 된 계기가 있다면 무엇인가요?
주변에서 디자인 부탁을 하는 경우가 많아 무료로 디자인을 해주곤 했었습니다. 그러던 중 몇년 전 친구가 소개해줬던 오투잡이란 사이트가 생각이 났습니다. 판매글을 올리는데 부담도 없고, 일단 한번 판매해보자 라는 생각에 판매글을 올리게 되었습니다.
어떤 장점 때문에 오투잡에서 재능을 판매하나요?
오투잡의 큰 장점은 ‘홍보’인 것 같습니다. 디자인프리랜서로 일하는 사람들을 보면 부러웠지만 쉽게 다가가기 힘들었습니다. 어디서 어떻게 진행해야 되는지 몰랐고, 디자인 프리랜서로 업무하시는 분들이 많기 때문에 오더를 따기가 쉽지 않았습니다.

오투잡에선 나를 필요로하는 사람들이 많아졌고, 내시간 내 스케쥴을 잘 활용해서 업무를 진행할 수 있으니 너무 좋은 것 같습니다. 그리고 믿을 수 있는 거래라고 생각합니다. 판매자와 구매자의 중간역할을 톡톡히 해주는 오투잡 덕에 마음편히 거래를 할 수 있는 것 같습니다.
수입은 얼마 정도인가요?
간단한 업무일 경우 5000원부터 시작합니다. 많을 경우 30~50만원 천차만별입니다.
본업이 있는 상태에서 투잡으로 활동하는 것인가요?
현직 디자이너지만 회사 시스템상 시간적인 여유가 있어 투잡으로 활동하고 있습니다. 부족한 시간은 퇴근 후 업무를 하기 때문에 큰 부담은 없는 것 같습니다.
투잡으로 병행하려면 힘들지 않나요?
화장품 회사에서 그래픽디자인을 담당하고 있습니다. 회사업무와 오투잡 업무가 겹칠 경우 조금 힘이들지만 급한 수정요청이 아닌 이상 집에서도 업무를 하기 때문에 괜찮은 것 같습니다.
투잡을 병행하는 데 있어 알려줄 만한 팁, 노하우는 무엇인가요?
시간 활용을 잘해야되는 것 같습니다. 업무요청이 올 경우 스케쥴에 대해서 충분히 설명하고, 조율하고 시간계획을 잘 짜는게 투잡의 팁 인 것 같습니다.
본인이 파워셀러가 된 이유를 뭐라고 생각하나요? 매출 증대를 위한 노하우나 고객만족을 위한 노하우가 있다면 알려주세요.
구매자 분들이 가장 중요하게 생각하는 것은 ’빠른답변’, ‘합리적인가격’, ‘디자인 퀄리티’ 라고 생각합니다. 합리적인 가격과 디자인 퀄리티 모두 만족 시켜 드릴 수 있어서 기존에 저에게 구매하셨던 분들은 지속적으로 업무를 주시는 것 같습니다. 그리고 ‘친절과 무한수정’ 입니다. 꼼꼼하신 분들은 작은거에도 엄청난 수정을 요구하실 때가 있습니다. 무리한 수정 요청이 와도 항상 친절하게 답변해드렸고, 수정요청이 많아도 구매자분들의 마음에 들어야 저도 업무가 마무리 된 것 같아 마음에 드실 때까지 해드렸습니다.
주로 어떤 분들이 이용하시나요?
다양한 연령층이지만 주로 사업하시는 분들, 직장에 다니시는 분들이 요청을 많이 하시는 것 같습니다. 중간중간에 과제나, 본인이 필요해서 요청하시는 분들도 많지만 주로 사업하시는 분, 직원분들이 요청을 많이 하십니다.
재능을 판매하면서 느끼는 좋은 점은 무엇인가요?
내가 한 디자인이 구매자 분들이 만족해 했을 때 가장 행복하고 보람을 느낍니다. 그리고 용돈같이 쏠쏠하게 들어오는 금액들이 모이고 모여 월급이 되었을 때, 또한 너무 행복합니다. 여행도 다니고, 맛있는 것도 먹고 오투잡 판매자가 된 후로 마음의 여유가 생겼습니다.^^*
재능 거래에 대해 어떻게 생각하나요?
구매자와 판매자가 둘다 윈윈 할 수 있는 거래라고 생각합니다.
재능 거래 시 주의할 점은 무엇인가요?
스케쥴을 잘 짜야한다는 점 입니다. 들어오는 일들을 스케쥴없이 받아버리면 구매자 분들에게 퀄리티도, 마감약속도 못 지키게 되어 시간관리를 철저히 해야 한다고 생각합니다.
재능 거래 시 겪은 어려운 점과 이를 극복한 노하우는 무엇인가요?
구매자분이 전달하는 내용과 제가 이해하는 부분이 서로 맞지 않아 디자인을 여러번 번복했을 때 어려웠던 것 같습니다. 그래서 초반에 서로 충분한 대화가 이루어져야 한다고 생각합니다.
오투잡에서 향후 계획이나 목표가 있다면 말씀해주세요.
구매자분들의 만족을 위해서 열심히 배우고 노력하는 디자이너가 되는 것이 목표이고 개인적으로 월에 고정적으로 넣는 적금이 있습니다. 목표금액까지 모이면 여행갈 생각입니다.
